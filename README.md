# Waw Player From SDCard Wtih PIC18F2550

* SdCard Fat Library : Petit Fat Library v0.02a
* Compiler ide : Mplab X
* Input Supply Range : 5 - 15VDC
* Audio Amplifier : TDA2005
* Audio Output Power : 20W
* Audio Creation Method : PWM
